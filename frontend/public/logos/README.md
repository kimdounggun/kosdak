# 로고 이미지 저장 폴더

## 사용 방법

1. **이미지 파일 준비**
   - 네이버 금융, 다음 금융 등에서 종목 로고 이미지를 다운로드
   - 배경이 투명한 PNG 형식 권장
   - 권장 크기: 200x200px 이상 (정사각형)

2. **파일명 규칙**
   - 파일명: `{종목코드}.png`
   - 예시:
     - `005930.png` (삼성전자)
     - `293490.png` (카카오게임즈)
     - `035420.png` (NAVER)

3. **파일 저장**
   - 이 폴더(`frontend/public/logos/`)에 저장
   - Next.js가 자동으로 `/logos/{종목코드}.png` 경로로 서빙

4. **우선순위**
   - 1순위: 로컬 파일 (`/logos/{종목코드}.png`)
   - 2순위: DB의 `logoUrl` 필드
   - 3순위: Fallback 아이콘 (종목명 첫 글자)

## 이미지 다운로드 방법

1. **네이버 금융**
   - https://finance.naver.com/item/main.naver?code={종목코드}
   - 종목 페이지에서 로고 이미지 우클릭 → "이미지 저장"

2. **다음 금융**
   - https://finance.daum.net/quotes/A{종목코드}
   - 종목 페이지에서 로고 이미지 우클릭 → "이미지 저장"

3. **기업 공식 홈페이지**
   - 회사 홈페이지에서 로고 다운로드
   - 보통 `/favicon.ico` 또는 `/logo.png` 경로에 있음
